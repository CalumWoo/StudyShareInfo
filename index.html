Problem Statement

Students struggle to find reliable , free revision materials for specific subjects and topics.

StakeHolders

  • Students

    • Students would be able to do everything

  • Teachers

    • Teachers would be able to upload resources (and view the most popular resources?)

What is my project about ?

Here are the basic features which my project may include:

  • User registration and logic system
  • Upload resources
    • images
    • videos
    • links
    • documents
  • Browse resources
    • Search by exact name
    • Search by subject tag
  • Upvote and downvote system for uploaded resources
  • View and download content
