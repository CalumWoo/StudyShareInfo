What is my project about ?

My project is a website that allows students to share and access
free study resources. It aims to provide a platform where students
can upload, browse, and download educational materials such as
images, videos, links, and documents.

  • User registration and logic system
  • Upload resources
    • images
    • videos
    • links
    • documents
  • Browse resources
    • Search by exact name
    • Search by subject tag
  • Upvote and downvote system for uploaded resources
  • View and download content

How is my project decomposited?

  • Authentication : Register, Login, Logout , Delete Account
  • Upload System : Add files or links with descriptions and subject tags
  • Database : Store users, resources, and votes
  • Browsing : Filter resources by subject and sort by popularity
  • Voting System : Upvote/downvote with one vote per user
